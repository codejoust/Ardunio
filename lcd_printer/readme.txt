Dead-Simple LCD Printer.

Connect an LCD to the ardunio (remember to adjust the pins in the code accordingly),
    then just run the python script... type messages to show.
   
A hello-world for the LiquidCrystal library that's a lot more fun!

A great tutorial for wiring an LCD can be found at 
    http://www.ladyada.net/learn/lcd/charlcd.html
    
Happy Hacking!
